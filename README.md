# Animations

A collection of interactive animations built with HTML, CSS, and JavaScript. Inspired by various web animation templates and customized for specific use cases.

## Included Animations

- **RGB Color Shift** - WebGL chromatic distortion
- **Dental Studio Reveal** - Animated brand reveal
- **Editorial Hover Showcase** - Interactive hover effects
- **Light Scroll Glow** - Scroll-based light animations
- **Data Visualization Scroll** - Scroll-triggered visualizations
- **Component Card Reveal** - Card reveal animations

## Quick Start

Open any `index.html` file in your browser to view the animation.

animation-name/
